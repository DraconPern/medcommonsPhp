HealthBook Installers

Windows (XP and Vista Only)

The current version of HealthBook Editor For Windows is 1.0d22 released on July 31, 2008.

Installation procedure:
  • Download (18MB) and then run Windows installer.
  • Refresh the HealthURL Viewer. For Internet Explorer you will need to restart the browser.

Other Operating Systems

There are currently no HealthBook editors for Linux or MacOSX.
