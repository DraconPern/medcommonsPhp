DDL Plugin for Osirix
The MedCommons DDL Plugin for OsiriX helps users of this popular 3D Radiology Workstation to connect to any HealthURL for the purpose of teleradiology, collaboration and archival storage. About OsiriX and the DDL Plugin
DDL
The MedCommons DICOM Data Liberator (DDL) enables upload and download of diagnostic images in the industry-standard DICOM format. DDL can be used to upload DICOM CDs still used by many hospitals and imaging centers. DDL can also be configured to connect directly to a facility's scanners and workstations. Install DDL
HealthBook PHR Editor
The MedCommons HealthBook PHR editor enables the editing of personal health records (PHR) stored in the industry-standard Continuity of Care Record (CCR) file format. Install HealthBook
