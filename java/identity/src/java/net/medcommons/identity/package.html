The identity webapp handles user login and registration for MedCommons users.

Once a user is validated, the user's web browser is redirected to the gateway server that actually has the user data.

New users can create an account using the Register servlet.

Existing users can login using the Login servlet. Hospitals or other health-care organizations can have their users automatically log in, if they support SAML. Our SAML server validates their users, then forwards to our Login servlet.
