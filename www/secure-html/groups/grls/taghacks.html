TagHacks - set registry fields from Tags

put the tag xxx: in the Comment Field of the TEPR Registry to affect the following other registry fields

    The following TagHacks are currently defined
  • pfn: PatientFamilyName
  • pgn: PatientGivenName
  • pid: PatitentIdentifier
  • pis: PatitentIdentifierSource
  • sprid: SenderProviderId
  • rprid: ReceiverProviderId
