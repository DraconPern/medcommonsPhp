Xform Support for Trackers

This note details progress towards the implementation of Xforms within MedCommons Trackers.

The Tracker Input Message Format is an XForms 'Instance'

Because MedCommons is so keen on standards, and there really isn't any standard, we can use XML and Xforms to define a Message Format for Trackers. This will allow any forms system, PDA, or lab system that utilizes Xforms to send Tracker messages into a MedCommons account

Here is a sample Tracker Message:
38292038291938392-10305066999-trackermsg.xml Here is another:
38292038291938392-10305066434-trackermsg.xml

You can send either of these files to the Tracker Xforms handler at /trackers/xform/handleTrackerMsg.php?file=
