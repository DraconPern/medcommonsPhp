Trackers Help beta release 0.1

Overview of Trackers

You must be logged on to use Trackers

Tracker Panel

The main tracker panel shows each of the trackers you have selected for display. To change the trackers on display, or to change any of their properties, use the Tracker Editor.

Tracker Input

You can add new values to an existing tracker by clicking anywhere on a tracker graph. There are two ways to add data:

  • Manual Entry - type the new value for your tracker.
  • CSV File Upload - the file must have a list of comma separated values

Tracker Editor

Use the editor to select and add trackers to your panel.
